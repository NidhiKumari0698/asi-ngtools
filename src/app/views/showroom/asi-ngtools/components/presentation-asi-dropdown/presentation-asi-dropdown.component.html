asi-dropdown

This component allow you to display a dropdown based on a component position

he's used on asi-select / asi-autocomplete / asi-calendar

Open dropdown
Here is the dropdown : You can add everything you want here
Name Description
[open] Boolean indicating if the dropdown should be opened or not
[calculWidth] Boolean: true if the dropdown should have the same width than its relative node. By default: true.
[relativeTo] The elementRef the dropdown refers to. Can be retrieve on html by #id or by a @ViewChild().
(onClose) Event triggered when the dropdown is closed when clicking outside
