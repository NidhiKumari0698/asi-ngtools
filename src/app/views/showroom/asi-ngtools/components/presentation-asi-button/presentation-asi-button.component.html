Name Description
@asi-button-background-color Background color of the button
@asi-button-label-color Label color of the button
@asi-button-border Border of the button
@asi-button-border-radius Border radius of the button
@asi-button-outline-border Border of the button when focused
@asi-button-hover-border Border of the button when hovered
@asi-button-hover-background-color Background color of the button when hovered
@asi-button-bordered-background-color Background color of the bordered button
@asi-button-bordered-label-color Label color of the label in a bordered button. Default value = @asi-button-label-color
@asi-button-bordered-border Border of the bordered button. Default value = @asi-button-border
