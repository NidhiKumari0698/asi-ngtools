asi-fa-icon

import AsiNgToolsModule or AsiFaIconModule
Name Description
[icon] Name of the fa-icon (Font Awesome), rendered as a css class
[label] Component label
[labelPosition] Component label position in 'top', 'left', 'right', 'bottom
[submit] [boolean] true for a submit type, false for a button type
[size] Size of the fa-icon within '1x', 'lg', '2x', '3x', '4x', '5x'
[disabled] [boolean] true to disable click on the fa-icon, false else
[tooltip] Text to display on a tooltip
[tooltipPosition] Tooltip position within 'rigth', 'left', 'top', 'bottom', bottom by default
[noClickOnLabel] Disable click on label only
(onClick) Event fired when the icon or label is clicked
Name Description
@asi-fa-icon-label-color Label color of the fa-icon
@asi-fa-icon-border Border of the fa-icon
@asi-fa-icon-outline-border Border of the fa-icon when focused
@asi-fa-icon-color Color of the label
