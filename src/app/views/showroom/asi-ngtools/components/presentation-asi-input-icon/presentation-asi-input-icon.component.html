Name Description
[label] Component label
[labelPosition] Component label position in 'top' , 'left' , 'right' , 'bottom
[icon] Component icon (font awesome)
[iconPosition] Component icon position in 'left' , 'right'
[placeholder] Allow you to define a placeholder when the input is empty
[pattern] Define a 'RegExp' pattern to respect while typing
[maxlength] Define the max length of the input
[number] If true the input only accept number by respecting pattern : "^-*[0-9,\.]*$"
[type] Input type in 'text', 'password'
(iconClicked) Event emitted when the icon is clicked
