asi-code-viewer

import AsiNgToolsModule or AsiCodeViewerModule This component is an integration of the Prism library, see : http://prismjs.com In your package.json add the dependency : "prismjs": "1.6.0" In your angular-cli.json add in scripts : "../node_modules/prismjs/prism.js", In your angular-cli.json add in styles : "../node_modules/prismjs/themes/prism.css" For specifique language you must add scripts : example for typescript add "../node_modules/prismjs/components/prism-typescript"
Name Description
[language] Language to display : 'html', 'typescript', 'css' etc... see primjs documentation
[fromUrl] Url of the file to display
[value] Display a value
