asi-code-viewer

Allow you to bind html code from a file or simple text value

This component is an integration of the Prism library, see : http://prismjs.com In your package.json add the dependency : "prismjs": "1.6.0" In your angular-cli.json add in scripts : "../node_modules/prismjs/prism.js" In your angular-cli.json add in styles : "../node_modules/prismjs/themes/prism.css" For specifique language you must add scripts : example for typescript add "../node_modules/prismjs/components/prism-typescript.js"
